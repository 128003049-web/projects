 Sorting Algorithm Comparator

 Description:
The Sorting Algorithm Comparator is a simple C++ program that compares the execution time of different sorting algorithms. The program generates random numbers and sorts them using multiple algorithms, then displays the time taken by each algorithm.

 Features:
- Implements multiple sorting algorithms
- Compares execution time of each algorithm
- Uses randomly generated input data
- Displays performance results

 Algorithms Implemented:
- Bubble Sort
- Selection Sort
- Insertion Sort

 Technologies Used:
- Programming Language: C++
- Concepts Used: Arrays, Functions, Time Measurement

 How to Run
1. Compile the program
   g++ sorting_comparator.cpp

2. Run the program
   ./a.out

 Example Output:
Bubble Sort Time: 0.004 seconds  
Selection Sort Time: 0.002 seconds  
Insertion Sort Time: 0.001 seconds

 Learning Outcomes:
- Understanding different sorting algorithms
- Comparing algorithm performance
- Measuring execution time in C++
